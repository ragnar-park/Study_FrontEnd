- 도달이 예상되는 엔드포인트에 대한 핸들러 생성
- 핸들러에서 응답을 보낼 서버를 생성
- setupsTests 파일을 업데이트해서 테스트전 서버가 요청을 리스닝하게 설정

### getAllByRole
- 일치하는 항목을 두 개 이상 검색
- axios 호출이 반환된 후에 이미지가 비동기식으로 채워져 동작하지 않음
 
### await findAllByRole
- 일반적으로 비동기식 업데이트가 필요하다면 사용
- DOM에 비동기식 요소가 나타날 때마다 사용
  - 그렇지 않으면 해당 요소가 나타날 떄까지 테스트가 진행되지 않음
